# react-loadingmask
react-loadingmask

## Install

```
npm i --save react-loadingmask
```

## Usage
```
import {LoadingMask} from 'react-loadingmask';
import 'react-loadingmask/dist/bundle.css';

<LoadingMask
  loading={true}
  loadingText={"loading..."}>
  <div style={{width:500,height:300}}>
    Compoment You want to display
  </div>
</LoadingMask>
```

## loading
true||false

## loadingText
text to show
